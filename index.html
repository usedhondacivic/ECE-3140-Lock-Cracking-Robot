Welcome to ECE3140 / CS 3420 GitHub Pages!

The repo is set up so you can easily edit the page using the online Markdown editor from GitHub. In order to access it

  • Go to "settings" in the repo
  • Scroll down to the section with GitHub pages
  • Press the "Launch Automatic Page Generator"

That should allow you to update (and remove) this text.

You are also welcome to use any other method to update/generate your course page. In order to update the material, push to the "gh-page" branch. Also note, that while the repo is private the page content is public.
