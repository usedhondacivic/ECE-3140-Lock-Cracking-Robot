Introduction

System Overview

The CNC Master Lock Cracker is a robot designed to efficiently find the combination to any master lock. First, place the lock in the lock holder and ensure the dial manipulator is seated on the dial. Rotate the dial so that the lock is at 0, then press the button on the microcontroller. The lock cracker will begin trying combinations, go make some coffee and come back to find it unlocked.

Progress to date:

I have completed all of the hardware and wiring for the project, and have written the stepper driver. It currently can unlock a lock when given its combination, but I still need to write the servo driver to automatically pull the shackle before brute forcing can begin.
